再生

episode の埋め込み再生はすぐ可能です。まずはログインなしでも触れる UI にしています。

次の段階

Spotify ログイン

あとから OAuth を足して、ユーザーごとの状態や保存処理を追加できます。

別実装が必要

Transcript コピー

transcript 本文はこの埋め込み iframe から直接は読めません。既存の Chrome 拡張をつなぐ方が現実的です。

次に足すなら

1. Spotify ログイン

  • Flask に callback route を追加
  • Spotify App の client id / secret を env に保持
  • ユーザー情報やプレイ対象の保存を可能にする

2. transcript 補助

  • 別タブで open.spotify.com の episode ページを開く
  • Chrome 拡張で transcript を抽出
  • 必要ならこのサイトに貼り戻して copy UI を出す

試作メモ

transcript をこのサイトに出すには、 open.spotify.com 上の DOM を読む経路を別途つなぐ必要があります。
